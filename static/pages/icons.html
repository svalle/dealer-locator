Using viewBox*

{{svg id="footer-twitter" className="acr-icon" viewBox="0 0 26 26"}}

*The viewBox property is required when you need to autoscale the icon (e.g. width: 100px; height: auto;), otherwise you have to set both width and height. Notice that each SVG icon has it's own viewBox value, that you can check on it's source code. If you set the specific value for your icon you can make it autoscale. This shouldn't be necessary if the icon is square since you'll be able to set your width and height via css which is the default way to do it. (e.g. width: 100px; height: 100px;)


New icons should be added from BOX folder only. SVG should be manually checked for unwanted inline style tags. Src folder for adding icons using default gulp task is: HTML/src/assets/svg/systems/*.svg


- The id is the name of the file, that should be the one provided by creative.
- The "acr-icon" class is the default style for an icon, you can add a new state on icons.scss or in your component sass if it is too customized
